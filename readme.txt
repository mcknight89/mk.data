This repository serves as a data store primarily for our datatoolkit.net website. It aggregates data from various sources for our own use, making it available publicly.

## Format
The data is provided in CSV and JSON formats.

## Data Usage
The data comes from various sources and is provided here for public use without any warranty.

You are free to use the data in this repository for personal, academic, and commercial purposes. Please note that while efforts have been made to ensure data accuracy, we do not guarantee the completeness or correctness of the data. Use the data responsibly and at your own risk.

## Contact
If you have any questions, feedback, or encounter issues related to the data in this repository, please feel free to open an issue on this GitHub repository.
